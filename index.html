You are not the guy, you are the level!

Stupax is a funny sidescroller (jump and run game) where you do not control the character, but the level. You control a movable platform to help the guy (namely Stupax) to get through the levels.

Play game right here!



Features:
  • Physics engine
  • Gamepad controls (using the Gamepad API)
  • Level editor
It is written purely in JavaScript using BabylonJS. So it runs in every browsers (Chrome or Chromium recommended due performance reasons).
